## Requirements

**Python** >= 3.6

Update your environment for instance by editing your **`.profile`** home file:

```
export PYTHONPATH="$PYTHONPATH:$HOME/pre.di.c/bin:$HOME/pre.di.c/clients/bin"
export PATH="$PATH:$HOME/pre.di.c/bin"
```
